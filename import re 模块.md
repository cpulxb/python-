## import re 模块

### re.match()
从字符串的起始位置开始匹配，匹配失败返回None,匹配成功返回对象

```python
import re #导入python标准库内置模块re
text='姓名:张三 生日:1999年1月1日 毕业日期:2020年9月1日'
t1=re.match('姓名.*生日:(\d{4}).*毕业日期:(\d{4}).*',text)
print(t1)#打印出来的是对象
print(t1.group())#将匹配到的值全部打印出来
print(t1.group(1),t1.group(2),t1.groups())#t1.group(i),是分组匹配，将某组的值打印
```
运行结果：
```
<re.Match object; span=(0, 33), match='姓名:张三 生日:1999年1月1日 毕业日期:2020年9月1日'>
姓名:张三 生日:1999年1月1日 毕业日期:2020年9月1日
1999 2020 ('1999', '2020')
```

### re.search()
扫描整个字符串并返回第一个成功的匹配
```python
t2=re.search('生日:(\d{4}).*日期:(\d{4})',text)
print(t2)#打印出来的是对象
print(t2.group())#打印pattern中的内容
print(t2.group(1))#打印括号中的值(分组后第一个组的值)
print(t2.group(2))#拿到第二组的数据
```
```
<re.Match object; span=(6, 28), match='生日:1999年1月1日 毕业日期:2020'>
生日:1999年1月1日 毕业日期:2020
1999
2020
```



### re.findall()
re.findall()一次能查找所有符合条件的内容,而re.research()一次能查找第一个成功匹配的
```python
t3=re.findall('生日:(\d{4})',text)
print(t3)#可以直接打印,因为返回的是一个列表
print(t3[0])#取出列表中的第一个值
t4=re.findall('生日:(\d{4}).*日期:(\d{4})',text)
print(t4)
print(t4[0][0])
```
运行结果：
```
['1999']
1999
[('1999', '2020')]
1999
```

```python
text1='''
姓名:张三 生日:1999年9月1日 毕业日期:2020年9月1日
姓名:李四 生日:2000年5月6日 毕业日期:2020年9月1日
姓名:王五 生日:1998年4月1日 毕业日期:2020年9月1日
'''
t4=re.findall('生日:(.*?) 毕业日期',text1)#.*是匹配任意数量的任意字符串,是贪婪模式,.*?是非贪婪模式
print(t4)
```
运行结果:
```
['1999年9月1日', '2000年5月6日', '1998年4月1日']
```

### re.sub()
替换函数,将函数某些值替换
```python
text2="0566-32323232"
t7=re.sub('^\d{4}','0755',text2)#将开头四个数字替换成0755
print(t7)#打印出替换后的内容
#显然用replace函数也可以完成替换,但0566换成了其他的数字，还要改变程序,而re.sub()利用正则表达式替换则更加灵活
text3='0566-32323232'.replace('0566','0755')
print(text3)
```
运行结果:
```
0755-32323232
0755-32323232
```
