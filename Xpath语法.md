### Xpath语法

```
XPath语法:XML Path Language是一门在XML和HTML文档中查找信息的语言，可以用来在XML和HTML文档中对元素和属性进行遍历
```

#### 语法

| nodename | 选取此结点的所有子节点                                   |
| :------- | -------------------------------------------------------- |
| /        | 从根节点选取                                             |
| //       | 从匹配选择的当前结点选择文档中的结点，而不考虑它们的位置 |
| .        | 选取当前结点                                             |
| ..       | 选取当前结点的父节点                                     |
| @        | 选取属性                                                 |
